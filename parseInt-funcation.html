Explict Type Casting | parseInt Funcation

parseInt Funcation with Integer value

document.write(parseInt(24)); //24

document.write(parseInt(12 + 12)); //24

document.write(parseInt("12") + 12); //24




parseInt Funcation with Float value

document.write(parseInt(3.142)); //3

document.write(parseInt(3.7 + 3.7)); //7

document.write(parseInt("24")) //24

document.write(parseInt("2" + "4")) //24

document.write(parseInt("22sometext")) //22

document.write(parseInt("1" + "2sometext")) //22

document.write(parseInt("somewhere23")) //NaN

document.write(parseInt("somewhere23text")) //NaN

document.write(parseInt(true)) //NaN

document.write(parseInt('')) //NaN
